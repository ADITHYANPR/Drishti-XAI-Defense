# 🛰️ Drishti-XAI-Defense

Explainable Deep Learning framework for robust military asset recognition under Electronic Warfare degradation.

---

## 🎯 Objective
To develop a robust and explainable AI system capable of identifying military targets from aerial imagery in degraded, contested environments.

---

## 🔥 Key Features
- Synthetic Electronic Warfare (EW) Simulation
- Deep Learning Backbone (ResNet / ViT)
- Grad-CAM Explainability
- Robustness Benchmarking
- Edge Deployment Optimization

---

## 🧠 System Pipeline
1. Input Image
2. Synthetic Noise Injection
3. Target Classification
4. Explainability Map
5. Performance Evaluation

---

## 🛡 Research Focus
- Trustworthy AI in Defense
- Robustness Under Noise
- Adversarial Resilience (Upcoming)
- Edge AI Deployment
